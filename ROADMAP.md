# Roadmap

Ideas and planned features for Gardener. Contributions welcome!

## Planned

- [ ] Production charts — visualize egg/honey/meat trends over time
- [ ] Breeding tracker — offspring, lineage, breeding cycles
- [ ] Care reminders — scheduled tasks for vaccinations, deworming
- [ ] Seed saving calendar — when to save seeds by plant
- [ ] Water usage tracking — per bed irrigation logging
- [ ] Multi-user support — shared gardens with backend auth
- [ ] Photo attachments — add photos to journal entries and harvests
- [ ] Print-friendly bed layouts — export bed plans as PDF
- [ ] Companion planting matrix — visual grid of all relationships
- [ ] Integration with weather APIs — automatic frost/watering alerts

- [x] 45+ plants with custom SVG icons

- [x] Companion/antagonist planting validation
- [x] Monthly self-sufficiency calculator (plants + animals)
- [x] Livestock management (chickens, ducks, rabbits, bees)
- [x] Health tracking (vaccinations, deworming, illness, loss)
- [x] Feed tracking with cost analysis
- [x] Seed inventory with viability tracking
- [x] Soil management (pH, N-P-K, amendments)

- [x] Annual food plan with crop targets + animal products
- [x] Expense tracking with 9 categories and ROI

- [x] Succession planting generator
- [x] Full backup/restore (15 data types)

- [x] Mobile-responsive with bottom navigation
